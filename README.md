# Ubuntu 20.04

### NodeJS 18.18.0

- Create folder

```
mkdir example
```

```
cd example
```

- Install module

```
npm install -g pm2
```

```
pm2 start var/www/htlml/project/location.js
```

```
pm2 save
```

```
pm2 list
```

```
pm2 startup
```

- after `pm2 startup`, copy the code
- running the code in terminal
- done
